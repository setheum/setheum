#  Swap Module

## Overview

Ethical DeFi's Built-in decentralized exchange module, the swap mechanism refers to the design of `Uniswap V3` with additional features and functionalities that makes  unique. In addition to being used for trading, DEX also participates in `liquidation`, which is faster than Liquidation By Auction when the liquidity is sufficient.
