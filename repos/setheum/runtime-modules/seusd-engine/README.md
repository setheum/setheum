# Setheum USD Engine Module

## Overview

Provides USD-Pegged `Setheum USD (SEUSD)` Stablecoin on Ethical DeFi.

The Setheum USD is Ethical DeFi's USD-pegged stablecoin, built on the `` (Ethical Collateralised Debt Position), the Pegged Stablecoin Protocol of the Ethical DeFi Suite is inspired by MakerDAO's design, natively built on the Setheum Network with Zero-interest loans with optimisations and native liquidation protocols which include our on-chain built-in DEX , a native auction system, as well as an on-chain native liquidation protection mechanism. The SetheumUSD is over-collateralised and multi-collateralised.
